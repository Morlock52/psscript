# How-to guides

How-to guides provide task-focused, step-by-step instructions.

![Deployment flow](/images/graphics/deployment-flow.svg)

## Common tasks

- Deploy or upgrade the platform.
- Configure authentication and access controls.
- Maintain infrastructure and data integrations.
- Use the Explain drawer and backfill cmdlet cards with Command Enrichment: `COMMAND-EXPLAIN-AND-ENRICHMENT.md`.
- Use the modern Script Editor (command palette, problems, versions/diff): `SCRIPT-EDITOR.md`.
