Crawl4AI Vector Database

v0.2.0

A vector database integration for crawl4ai that enables semantic search and chat capabilities over crawled web content.

What is Crawl4AI Vector Database?

Crawl4AI Vector Database is a powerful tool that combines web crawling with vector embeddings to enable semantic search and chat capabilities over web content. It uses PostgreSQL with pgvector for storing and querying vector embeddings.

With the latest improvements, it now supports deep crawling with multiple strategies, browser profiles for authenticated crawling, and multiple embedding providers.

Key Features

Deep Crawling

Recursively crawl websites using BFS, DFS, or BestFirst strategies to build a comprehensive knowledge base.

Browser Profiles

Create and manage persistent browser profiles for authenticated crawling of login-protected content.

Multiple Embedding Providers

Generate embeddings using OpenAI or HuggingFace models with configurable dimensions and parameters.

Usage Examples

Deep Crawling

npm run crawl site https://example.com --strategy bfs --max-pages 50

Browser Profiles

# Create a profile
npm run crawl profile create my-profile

# Use a profile for crawling
npm run crawl url https://example.com --profile my-profile

Semantic Search

npm run search query "What is vector search?"

Deep Crawling

POST /api/webpages
{
  "url": "https://example.com",
  "deepCrawl": true,
  "strategy": "bfs",
  "maxPages": 50,
  "threshold": 0.7
}

Get Child Pages

GET /api/webpages/:id/children

Semantic Search

POST /api/search
{
  "query": "What is vector search?",
  "limit": 5
}

Embedding Service

const embeddingService = new EmbeddingService({
  provider: 'openai',
  model: 'text-embedding-3-small',
  dimensions: 1536
});

const embedding = await embeddingService.generateEmbedding("What is vector search?");

Deep Crawling

const crawlService = new CrawlService();

const result = await crawlService.crawlAndStore(url, {
  deepCrawlEnabled: true,
  deepCrawlStrategy: 'BFS',
  maxPages: 50,
  scoreThreshold: 0.7
});
