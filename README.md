# PowerShell Script Management Application

A comprehensive application for managing, analyzing, and utilizing PowerShell scripts with AI-enhanced capabilities.

## Project Structure

- **frontend/** - React/TypeScript UI with Monaco Editor and Tailwind CSS
- **backend/** - Node.js/Express API server
- **ai/** - OpenAI integration for script analysis
- **db/** - PostgreSQL with pgvector for structured and vector data
- **powershell/** - PowerShell Core integration modules

## Core Features

- Script repository with version control
- AI-powered script analysis and categorization
- Semantic search capabilities
- Role-based access control
- Interactive execution environment

## Script Categories

The system automatically categorizes scripts into 20 predefined categories including System Administration, Network Management, Active Directory, Security Tools, and more.
